Peak Finding Details

Peak finding is performed using MACS (Model-based Analysis for ChIP-Seq). Click here to view the website, or refer to the publication below:

  • Zhang et al. Model-based Analysis of ChIP-Seq (MACS). Genome Biol (2008) vol. 9 (9) pp. R137).

The input for MACS is the BAM (.bam) and BAM index (.bam.bai) files provided above. Default parameters are used for the most part. If you are re-running an analysis using MACS at the command line yourself, the automated analysis can be replicated as follows (explanations of all MACS parameters can be found here and the ACTUAL values of the parameters used in this analysis can be found at the top of the '.xls information on called peaks' file linked to below):

  • IP only
    • macs --treatment=<bam file for IP sample> --name=<sample name> --format=BAM --gsize=<effective genome size> --tsize=<size of sequence fragments> --bw=<sonication fragment size * 0.5> --wig --space=100
  • IP vs input
    • macs --treatment=<bam file for IP sample> --control=<bam file for INPUT sample> --name=<sample name> --format=BAM --gsize=<effective genome size> --tsize=<size of sequence fragments> --bw=<sonication fragment size * 0.5> --wig --space=100
